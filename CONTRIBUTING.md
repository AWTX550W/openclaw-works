## 🤝 如何贡献

### 报告 Bug

如果你发现 bug 或有功能请求，请先 [搜索 Issues](../../issues) 确认是否已存在类似问题。

如果没有，请[创建新 Issue](../../issues/new)，包含：
- **Bug 描述**：清晰的问题描述
- **复现步骤**：如何重现这个问题
- **期望行为**：你期望的结果
- **环境信息**：操作系统、Python/Node 版本等
- **截图**（如适用）

---

### 提交 Pull Request

1. **Fork 本仓库**
   - 点击右上角 "Fork" 按钮

2. **克隆到本地**
   ```bash
   git clone https://github.com/YOUR-USERNAME/openclaw-works.git
   cd openclaw-works
   ```

3. **创建分支**
   ```bash
   git checkout -b feature/AmazingFeature
   ```

4. **提交更改**
   ```bash
   git add .
   git commit -m "Add: AmazingFeature description"
   ```

5. **推送到分支**
   ```bash
   git push origin feature/AmazingFeature
   ```

6. **开启 Pull Request**
   - 访问 https://github.com/AWTX550W/openclaw-works/pulls
   - 点击 "New pull request"
   - 填写标题和描述
   - 等待审核

---

### 代码规范

- **Python**: 遵循 PEP 8，使用 4 空格缩进
- **Shell**: 使用 bash，添加注释
- **Markdown**: 使用 ``` 代码块，合理使用标题层级
- **Commit message**: 使用祈使句（"Add"、"Fix"、"Update"）
